# Platform-Specific Documentation

Guides for building and publishing to specific platforms.

## Available Platforms

| Platform | Directory | Description |
| -------- | --------- | ----------- |
| NixOS | [`nixos/`](nixos/) | Flake and traditional NixOS module setup, bridge networking, provisioning |
| Docker | [`docker/`](docker/) | Docker Compose for non-NixOS systems |
| Static / Self-Hosted | [`static/`](static/) | Serve the private demo from Apache, Nginx, or any web server |
