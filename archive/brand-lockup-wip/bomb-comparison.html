Analysis: What potrace loses vs the PNG

  • 3D body gradient — PNG has smooth radial shading (lighter center, darker edges). Potrace flattens to solid #1c1c1c.
  • Green ring highlight — PNG has a smooth arc. Potrace fragments it into jagged path segments. (_3/_4 tried to fix with manual circle strokes.)
  • Spark glow/bloom — PNG spark has radial glow and white-hot center. Potrace makes flat amber blobs.
  • Fuse collar detail — PNG has a 3D rounded collar with green+brown layers. Potrace loses the depth.
  • Drop shadow — PNG has subtle shadow behind the bomb body. Completely lost in all traces.

Recommended Path: Hybrid Rebuild

Use bomb-only_5.svg as the shape reference (best silhouette data), then rebuild the visual treatment with proper SVG primitives:

  • Bomb body → <circle> + <radialGradient>
  • Green ring → <circle> with stroke (already attempted in _3/_4)
  • Fuse collar → <rect> + <path> with gradients
  • Spark burst → <polygon> / <path> with glow filter
  • Shadow → <feDropShadow> or blurred circle

The animation file (brand-animation-phase1.html) already has a simplified bomb built from basic shapes — that's the right technique, just needs more detail to match the PNG.
